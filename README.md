# DetailWeb Studio

Site vitrine one-page premium pour **DetailWeb Studio** — agence web spécialisée dans la création de sites internet pour les garages et studios de detailing automobile.

## À propos

DetailWeb Studio aide les professionnels du detailing automobile (correction peinture, céramique, PPF, polish, interior detailing) à obtenir une présence en ligne à la hauteur de leur savoir-faire.

## Contenu du site

- **Hero** — Accroche principale, stats clés
- **Problème** — Pain points des detailers sans site performant
- **Expertise** — 6 piliers de l'agence
- **Réalisations** — Portfolio de 6 projets fictifs
- **Fonctionnalités** — 14 features incluses (bento grid)
- **Avant/Après** — Comparaison visuelle
- **Tarifs** — 3 formules (Starter, Professional, Enterprise)
- **Processus** — 5 étapes de travail
- **Témoignages** — 4 avis clients fictifs
- **FAQ** — 8 questions fréquentes (accordéon)
- **CTA Final** — Appel à l'action
- **Contact** — Formulaire + infos

## Stack technique

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts : Bebas Neue + DM Sans
- Aucune dépendance externe (pas de jQuery, Bootstrap, etc.)
- SVG inline pour toutes les icônes
- Mobile-first, responsive (768px / 1024px)
- IntersectionObserver pour les animations au scroll
- Compatible GitHub Pages

## Déploiement

Ce fichier `index.html` est directement compatible avec **GitHub Pages**.

Activer dans `Settings > Pages > Source: main / root`.
